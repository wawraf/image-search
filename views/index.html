Image Search Abstraction Layer

Part 4 of Free Code Camp Backend Challenges

User stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Examples:

  1. https://helix-sun.glitch.me/cats
  2. https://helix-sun.glitch.me/cats?offset=4
  3. https://helix-sun.glitch.me/recent

Output:

  • First two queries will throw array of pictures (actually only links + some extra data) concerning particular query/search (in this case "cats").
  • Last (third) query will always throw array with most 10 recent search strings. You can try it HERE.
